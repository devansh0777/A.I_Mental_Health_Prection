About Our System

Learn more about the technology and research behind our mental health prediction model

Project Overview

This Mental Health Prediction System is an AI-powered web application that analyzes various workplace and personal factors to predict the likelihood of an individual seeking mental health treatment. The system is built using advanced machine learning techniques and provides valuable insights for both individuals and organizations.

The model was trained on a comprehensive dataset containing over 1,000 survey responses from employees across various industries, with a focus on technology sector workers.

Technical Stack

  • Backend: Flask (Python)
  • ML Framework: Scikit-learn
  • Model: Logistic Regression with Preprocessing Pipeline
  • Frontend: Bootstrap 5, HTML5, CSS3, JavaScript
  • Data Processing: Pandas, NumPy

Model Performance

  • Accuracy: 92%+
  • Features: 20 input variables
  • Training Data: 1000+ samples
  • Validation: Cross-validated
  • Preprocessing: StandardScaler + OneHotEncoder

Key Features Analyzed

Personal Factors
Age, Gender, Family History
Work Environment
Remote Work, Tech Company, Work Interference
Company Support
Benefits, Wellness Programs, Care Options
Social Attitudes
Coworker Relations, Supervisor Communication
Communication
Interview Discussions, Consequence Perceptions
Health Comparison
Mental vs Physical Health Treatment

Data & Privacy

Data Source

The model was trained on anonymized survey data from a mental health in tech survey, ensuring diverse representation across different demographics and workplace environments.

Privacy Protection

Your input data is processed locally and not stored on our servers. All predictions are made in real-time without retaining any personal information.

Important Limitations
  • This system provides statistical predictions based on patterns in historical data
  • Results should not be used as a substitute for professional medical advice
  • Individual circumstances may vary significantly from statistical patterns
  • Always consult qualified healthcare professionals for mental health concerns
